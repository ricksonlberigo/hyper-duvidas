Acessando o Sistema

Para acessar o sistema, insira suas informações de login nos campos a seguir: Chave de acesso, Usuário e Senha.

  • 1. Chave de Acesso - Cada empresa possui uma chave padrão que serve como seu identificador exclusivo. Normalmente, essa chave é fornecida pela administração da empresa.
  • 2. Usuário - Para acessar o sistema, você deve ter um cadastro de usuário. Se ainda não possui um, entre em contato com a administração da empresa ou com o departamento responsável pela criação de contas. Se você já possui um cadastro, informe o nome de usuário neste campo.
  • 3. Senha - Ao criar seu cadastro de usuário, você definiu uma senha. Insira sua senha neste campo.
  • 4. Ícone de Visualização - Se desejar visualizar a senha enquanto a digita, clique no ícone de olho localizado à direita do campo de senha.
  • 5. Lembrar Credenciais - Ative esta opção se desejar que o sistema lembre sua chave e usuário para futuros acessos. Por motivos de segurança, a senha não será lembrada.

Tela da Home

Após fazer o login no sistema, você será redirecionado para a tela inicial, a "Home". Nesta tela, você terá acesso a várias funcionalidades essenciais:

  • 1. Informações do Usuário - Aqui, você pode acessar as informações do seu perfil, incluindo detalhes pessoais, notificações e mensagens importantes.
  • 2. Top Vendedores - Esta seção apresenta os principais vendedores do mês atual. Você verá o ranking dos cinco vendedores que mais venderam, classificados de acordo com o valor total das vendas.
  • 3. Contato para Suporte - Se precisar de ajuda ou suporte em relação ao sistema, nossa equipe está pronta para ajudar. Basta clicar na opção "Contato" e escolher entre as opções de suporte via WhatsApp ou Skype.
  • 4. Blog da Empresa - Na parte inferior da página principal, você encontrará o blog da sua empresa. Este espaço é dedicado a compartilhar informações essenciais, atualizações e notícias relevantes para todos os colaboradores.
  • 5. Menu de Navegação Lateral - No lado esquerdo da tela, você verá o Menu de Navegação Lateral, que oferece acesso rápido às diferentes funcionalidades do sistema, como caixa, cadastros, transferências e muito mais. Ao passar o mouse sobre as opções, o menu se expandirá, exibindo as funcionalidades disponíveis.
  • 6. Menu de Janelas de Acesso - Conforme você acessa telas diferentes, o sistema cria guias ou janelas no menu superior para facilitar a navegação entre elas. Caso tenha várias janelas abertas, o sistema as organizará no menu "Minhas Janelas", localizado no início do menu superior.

É importante ressaltar que o botão "Fechar Todas as Telas" não encerrará aquelas que estiverem no meio do processo de adição, alteração de cadastros ou movimentações.

Acessando Telas

Para acessar as diversas telas disponíveis, siga as etapas a seguir:

  • 1. Posicione o cursor sobre o menu lateral de acessos.
  • 2. Selecione o tipo de acesso, seja um cadastro, movimentação ou outra opção que desejar.
  • 3. Escolha a tela específica que você deseja acessar. É simples assim!

Aprenda a Usar a Tela de Cadastros

Após acessar a tela de clientes, você verá os cadastros dispostos em ordem alfabética. Nessa tela, você conta com as seguintes opções:

  • 1. Adicionar Cadastro - Para criar um novo cadastro, clique no botão "+ Novo".
  • 2. Pesquisar Clientes - É possível buscar um cliente específico pelo nome, código do cadastro ou CPF, usando o campo de pesquisa.
  • 3. Visualizar Cadastro - Para acessar todas as informações de um cadastro, clique no ícone "Visualizar".
  • 4. Alterar Cadastro - Se precisar modificar informações de um cadastro, utilize o ícone de alteração localizado à direita do ícone de visualização.
  • 5. Excluir / Log de Atividades - Ao lado do ícone de alterar, há um menu com três pontinhos. Ao clicar nele, o sistema exibirá duas opções: "Excluir" e "Log de Atividades".
    • A opção "Log de Atividades" mostrará todas as alterações feitas após a implementação do cadastro.
    • A opção "Excluir" apagará permanentemente o cadastro. Ao excluir um cadastro, você também perderá todas as informações vinculadas a ele, como vendas e movimentações. Por isso, é recomendável inativar o cadastro em vez de excluí-lo para preservar esses dados.
  • 6. Paginação - Na parte inferior, à direita, estão as seções dos cadastros. O sistema, por padrão, exibirá 10 registros por vez. Caso você tenha mais de 10 cadastros, o sistema criará uma nova seção. Para navegar entre as seções, utilize as setas:
    • A seta "<" apontada para o lado esquerdo volta para a seção anterior.
    • A seta ">" apontada para o lado direito avança para a próxima seção.
    • As duas setas "<<" apontadas para o lado esquerdo levam à primeira seção.
    • As duas setas ">>" apontadas para o lado direito levam à última seção.
  • 7. Ordem de Exibição / Status - Caso prefira visualizar os cadastros em ordem diferente da alfabética, você pode selecionar a ordenação por código no menu que aparece ao final do campo de pesquisa . Além disso, é possível escolher quais status de cadastro deseja visualizar. Por exemplo, marcar a opção "Inativos" exibirá tanto os clientes ativos quanto os inativos, enquanto marcar apenas a opção "Ativos" mostrará apenas os clientes ativos.

Cadastro de Cliente

Ao acessar a tela de cadastro de clientes, você notará que ela está dividida em "abas" para organizar as informações. Vamos entender como cada aba funciona:

  • 1. Informações Principais - Nesta aba, você encontrará os campos destinados às informações fundamentais do cliente, como nome, CPF/CNPJ, cidade e outras informações essenciais.
  • 2. Dados Avançados - Na segunda aba, estão disponíveis informações complementares sobre o cliente. Aqui, você pode inserir dados como o nome da mãe, nome do pai, observações e outras informações detalhadas.
  • 3. Log de Atividades - A terceira aba, chamada "Log de Atividades", não contém campos a serem preenchidos. Ela desempenha um papel importante, registrando todas as alterações feitas no cadastro. Assim, essa aba armazena um histórico das modificações realizadas no cliente, permitindo o acompanhamento de todas as mudanças ao longo do tempo.

Informações Importantes

Preencher os dados de forma completa e precisa é de extrema importância, principalmente quando se trata da emissão de Notas Fiscais Eletrônicas (NF-e) ou Cupons Fiscais Eletrônicos (NFC-e).

  • 1. Inativar Cadastros - Durante o processo de cadastro de clientes, a função de inativar não estará disponível imediatamente. O sistema, por padrão, cria novos cadastros com a opção "ativo" marcada, evitando inativar clientes recém-cadastrados. Para tornar um cliente inativo, basta acessar o cadastro posteriormente e desmarcar a opção "ativo".
  • 2. Outros Endereços - É possível adicionar endereços secundários ao cadastro de clientes. Para isso, basta clicar na opção "outros endereços" localizada na parte inferior do cadastro. Isso permite incluir informações detalhadas de endereços adicionais conforme necessário.
  • 3. Campos Cruciais para Documentos Fiscais - Se você planeja transmitir Notas Fiscais Eletrônicas (NF-e) ou Cupons Fiscais Eletrônicos (NFC-e), certifique-se de que campos importantes estejam devidamente preenchidos. Esses campos incluem CPF/CNPJ, RG/Inscrição estadual, UF, endereço, número e CEP, uma vez que são validados durante a transmissão de documentos fiscais.

Cadastro de Produto

A tela de cadastro de produtos no sistema Hypersoft compartilha muitas semelhanças com outras telas de cadastro. Para adicionar um novo produto, siga o mesmo procedimento de acessar a tela de produtos e clicar em "+ Novo".

  • 1. Informações Principais - Ao acessar a tela de cadastro de produtos, você será direcionado para a primeira aba, chamada "Principal". Nesta aba, você encontrará as informações cruciais do produto, como nome, custo atual, EAN e outros detalhes importantes. É relevante observar que, caso um campo possua um ícone de informação ao lado de seu nome, isso significa que informações adicionais estão disponíveis. Você pode visualizá-las passando o mouse sobre o ícone de informação.
  • Uma informação fundamental é a necessidade de prestar atenção nos campos fiscais, especialmente para empresas que realizam transmissões de Notas Fiscais Eletrônicas (NF-e) e Cupons Fiscais Eletrônicos (NFC-e). Esses campos fiscais devem ser preenchidos com o auxílio do escritório contábil. Na aba "Principal," você encontrará os seguintes campos fiscais: Alíquota de Venda, Saída - Situação Tributária Pis/Cofins e Entrada - Situação Tributária Pis/Cofins.
  • 2. Configurações Adicionais - Na segunda aba, chamada "Configurações Adicionais," você pode acessar algumas configurações extras, como os tipos de venda permitidos, a necessidade de solicitação de números de série e outras informações fiscais.
  • Esta aba também contém campos fiscais que devem ser preenchidos com o auxílio do escritório contábil. Esses campos incluem CST, Alíquota, Redução, NCM, CEST, Natureza da Operação e Partilha do ICMS.
  • 3. Log de Atividades - A terceira aba, chamada "Log de Atividades", não contém campos a serem preenchidos. Ela desempenha um papel importante, registrando todas as alterações feitas no cadastro. Assim, essa aba armazena um histórico das modificações realizadas no cliente, permitindo o acompanhamento de todas as mudanças ao longo do tempo.

Contas a Pagar
  • 1. Introdução ao Contas a Pagar:
    • O Contas a Pagar fornece uma visão organizada das despesas pendentes, como faturas de fornecedores, contas de despesas e pagamentos de empréstimos.
    • Os usuários podem classificar e priorizar essas obrigações com base em critérios como prazos de vencimento, valor e fornecedor.
  • 2. Acesso à Tela de Contas a Pagar:
    • Acesse a tela de Contas a Pagar a partir do menu lateral.
    • Clique em "Financeiro" e, em seguida, em "Contas a Pagar" para acessar a tela.
  • 3. Adicionando uma conta a Pagar:
    • Para adicionar uma nova despesa, clique no botão "+ Novo".
  • 4. Filtrando Lançamentos:
    • Use o botão "Filtro" para filtrar os lançamentos.
    • Escolha opções de filtro, como loja, fornecedor e data, e clique em "Filtrar".
  • 5. Acesso Rápido:
    • Encontre atalhos para outras telas relacionadas ao Contas a Pagar.
  • 6. Status Despesas:
    • Verifique os valores totais para diferentes status, como Não Autorizado, Autorizado, Total a Pagar e Total Pago.
  • 7. Aba de Despesas Pendentes:
    • Visualize todas as despesas pendentes.
    • As despesas não autorizadas são marcadas em vermelho, e as autorizadas em azul.
    • Se tiver permissão, é possível dar baixa em lançamentos não autorizados.
  • 8. Funções na Aba Pendente:
    • Ao clicar nos três pontos, você pode autorizar, não autorizar, baixar, editar, visualizar, excluir ou verificar o log de atividades de um lançamento.
  • 9. Baixar/Outras Ações:
    • É possível selecionar vários lançamentos e dar baixa em todos de uma vez.
    • Também é possível autorizar ou desautorizar vários lançamentos simultaneamente.
  • 10. Aba de Despesas Pagas:
    • Visualize as despesas que já foram pagas.
  • 11. Funções na Aba Pagos:
    • Ao clicar nos três pontos, você pode verificar o log de atividades de um lançamento e estornar despesas pagas.
    • É possível estornar várias despesas pagas de uma vez.

Contas a Pagar - Lançamento
  • 1. Clique no Botão Novo:
    • Para criar um novo lançamento de Contas a Pagar, clique no botão "Novo" na tela de Contas a Pagar.
  • 2. Informando os Dados do Documento:
    • Após abrir a tela de lançamentos, forneça as informações do documento que deseja registrar.
    • Você pode inserir detalhes, como valor, data de vencimento, fornecedor e mais.
  • 3. Opções "Autorizar Baixa":
    • Importante observar que, se você deseja autorizar automaticamente o lançamento para baixa, vá até a aba "Avançado" e marque a opção "Autorizar Baixa".
    • Ao selecionar essa opção, o documento será autorizado para baixa assim que for registrado. Se não desejar autorizar o documento no momento do registro, simplesmente deixe essa opção desmarcada.
  • 4. Observações sobre Permissões:
    • Lembre-se de que, se não tiver permissão para dar baixa em documentos não autorizados, será necessário ter a permissão adequada atribuída ao seu usuário. Caso contrário, solicite assistência a um usuário com as devidas permissões.
  • 5. Preenchendo Informações de Pagamento:
    • Insira as informações relacionadas ao pagamento, como a forma de pagamento, conta bancária, e outros detalhes relevantes.
    • Após preencher todas as informações, clique em "Confirmar".

Nova Venda

A tela "Nova Venda" permite que os funcionários da loja registrem transações de venda de maneira eficiente e simplificada. Esta interface é projetada para que os vendedores possam selecionar produtos, aplicar descontos, adicionar itens ao carrinho e concluir a venda de forma eficaz. Siga os passos a seguir para realizar uma venda:

  • 1. Preenchimento das Informações Iniciais:
    • Na tela de venda, preencha as seguintes informações:
    • Empresa: Selecione a empresa onde a venda será realizada (o sistema geralmente traz a empresa principal associada ao usuário logado).
    • Data: Insira a data da venda (normalmente o sistema já preenche a data atual).
    • Vendedor: Escolha o vendedor que está realizando a venda (o sistema geralmente preenche o nome do vendedor com base no cadastro do funcionário logado).
    • Cliente: Digite o nome do cliente que está fazendo a compra.
    • Endereço: O sistema preenche automaticamente com o endereço principal do cliente selecionado. Se o cliente tiver outros endereços, é possível adicionar um novo endereço clicando em "Novo Endereço" após a seleção do cliente.
  • 2. Abas na Parte Inferior:
  • A parte inferior da tela possui as seguintes abas:
    • Produtos: Nesta aba, selecione os produtos a serem vendidos, insira a quantidade e, se necessário e permitido, ajuste o valor unitário. Clique em "Adicionar" para incluir o produto na venda. Se um produto requer ativação ou tem número serial, uma janela será aberta para preenchimento.
    • Entradas: Use esta aba para registrar devoluções ou vendas com trocas. Adicionar um produto aqui indica que o produto está retornando para o estoque e será descontado do valor da venda.
    • Serviços: Esta aba permite adicionar serviços realizados na venda.
    • Adicional: Insira observações relevantes sobre a venda, como o local onde o cliente retirará o produto.
  • 3. Descontos:
    • À direita da tela, você pode adicionar descontos ou acréscimos à venda. Selecione a opção desejada e especifique se o valor será aplicado como porcentagem ou valor fixo. Clique em "Aplicar" para incluir o desconto ou acréscimo na venda.
  • 4. Pagamentos:
    • Para finalizar a venda, clique em "Formas de Pagamento". Aqui, é possível adicionar uma ou várias formas de pagamento. Se o cliente realizar o pagamento com apenas uma forma, selecione a forma e o sistema aplicará o valor total automaticamente. Se o cliente utilizar várias formas de pagamento, insira o valor em cada forma até que o total seja atingido.
  • 5. Finalização da Venda:
    • Após inserir as informações da forma de pagamento, clique em "Finalizar". O sistema gerará o comprovante de venda.
    • Se a venda for interrompida antes da finalização, você pode clicar em "Cancelar" para limpar todas as informações e reiniciar o processo.
  • 6. Informações Importantes:
    • Alguns campos na tela de vendas possuem um ícone de lápis com uma seta para baixo. Ao clicar nesse ícone, um submenu é exibido com opções como "Novo", "Cadastro Completo" e "Limpar". Essas opções permitem a adição rápida de clientes ou acesso direto a informações completas do cadastro.

Pré-Venda

A tela de Pré-Venda é uma ferramenta essencial para os vendedores registrarem e processarem pedidos de clientes antes que a transação de venda seja finalizada. Embora siga o mesmo modelo da Venda Balcão, a principal diferença é que a Pré-Venda não é finalizada na própria tela. Em vez disso, a Pré-Venda é registrada e posteriormente finalizada na tela de Caixa.

  • 1. Na tela de Pré-Venda, preencha todas as informações relevantes da venda, incluindo detalhes do cliente, produtos, quantidade, preços e quaisquer descontos aplicáveis.
  • 2. Após preencher todos os dados da Pré-Venda, você não finaliza a venda nesta tela. Em vez disso, a venda é salva temporariamente e aguarda finalização na tela de Caixa.
  • 3. Vá para a tela de Caixa quando estiver pronto para concluir a transação. Na tela de Caixa, você seleciona a Pré-Venda correspondente e prossegue para finalizá-la.
  • 4. No processo de finalização, você escolhe a forma de pagamento, confirma os valores e, em seguida, clica em "Finalizar Venda" para concluir a transação e registrar o pagamento.

Consulta de Vendas

A tela de Consulta de Vendas é projetada para fornecer acesso e análise de informações essenciais sobre as vendas realizadas pela organização. Essa funcionalidade é utilizada por gerentes, membros da equipe de vendas e outros usuários para obter insights sobre o desempenho de vendas da empresa.

  • 1. Pesquisar Vendas - O principal recurso desta tela é a capacidade de pesquisar vendas. Você pode pesquisar vendas com base no número do pedido (Nº Pedido) ou pelo nome do cliente. Além disso, é possível aplicar filtros adicionais, como data e empresa, para refinar ainda mais os resultados da pesquisa. Basta clicar nos ícones de filtro à direita para definir seus critérios de pesquisa.
  • 2. Ações Disponíveis - Após encontrar a venda desejada, você tem várias ações disponíveis. Pode visualizar os detalhes da venda, imprimir informações relevantes ou, se necessário, excluir a venda. Para opções adicionais, clique nos três pontos (menu suspenso) para acessar o histórico de atividades relacionadas à venda.
  • 3. Iniciar uma Nova Venda - Se você deseja iniciar uma nova venda, basta clicar no botão "Novo" localizado na parte superior da tela. Isso abrirá a tela de vendas, onde você pode começar a registrar uma nova transação de venda.

Consulta de estoque

A tela de Consulta de Estoque é projetada para fornecer informações detalhadas sobre os níveis de estoque de produtos em sua empresa. Ela desempenha um papel crucial no eficiente gerenciamento de estoque, permitindo que você tenha uma visão completa e atualizada do status de seus produtos e recursos disponíveis.

  • 1. Filtrar - Para verificar os níveis de estoque dos produtos, você deve preencher os filtros necessários. Isso pode incluir a escolha de uma loja específica, seleção de produtos ou categorias, entre outros critérios. Depois de configurar seus filtros, clique no botão "Filtrar" para obter os resultados.
  • Observações Importantes:
    • Evite selecionar todas as empresas e todos os produtos de uma só vez, pois isso pode não funcionar adequadamente devido à quantidade de informações envolvidas. A consulta é mais eficaz quando focada em lojas ou produtos específicos.
    • O campo "Exibir" permite que você escolha se deseja visualizar informações de lojas ou produtos, mas não é possível exibir ambos ao mesmo tempo em certas configurações. Certifique-se de fazer escolhas compatíveis para evitar erros.
  • 2. Exportar Dados - Se desejar, é possível exportar os dados de estoque filtrados para uma planilha clicando no botão "Exportar". Isso pode ser útil para análises adicionais ou relatórios personalizados.

Instalações / Serviços - Funções

A tela de Instalações / Serviços é uma ferramenta fundamental para empresas que lidam com solicitações de serviços, instalações, manutenção e reparos. Ela simplifica o gerenciamento e controle dessas atividades, desempenhando um papel vital na organização operacional.

  • 1. Filtrar - Use o filtro para localizar ordens de serviço por número (N° OS) ou protocolo. Personalize ainda mais sua pesquisa selecionando loja, técnico, cliente, tipo de serviço e status.
  • 2. Adicionar Produto - Clique em "+Novo" para criar uma nova ordem de serviço e iniciar o processo.
  • 3. Ações - Na lista de ordens de serviço, você tem várias opções de ação, incluindo:
    • Visualizar - Utilize o ícone de um "olho" para examinar o conteúdo detalhado de uma OS.
    • Alterar - O ícone de um "lápis" permite que você faça modificações em informações específicas da ordem de serviço.
    • Três Pontinhos - Ao clicar nos três pontinhos, você pode imprimir uma OS que já foi fechada. Caso a OS esteja em andamento, é possível cancelá-la e também realizar a impressão se necessário.

Instalações / Serviços - Nova Ordem de Serviço

Para criar uma nova ordem de serviço na tela de Instalações / Serviços, siga estes passos:

  • 1. Clique no botão "Novo" na tela de Instalações / Serviços.
  • 2. Você será redirecionado para a tela de adição da ordem de serviço, onde deverá preencher as seguintes informações:
    • Loja - Indique a loja responsável pela OS. O sistema automaticamente preenche com a loja padrão do cadastro do usuário.
    • Técnico - Selecione o técnico encarregado de executar a OS. O sistema preencherá automaticamente com o técnico associado ao usuário logado, desde que esteja cadastrado como técnico no cadastro de clientes.
    • Cliente - Informe o cliente que solicitou o serviço.
    • Endereço - O sistema preencherá automaticamente com o endereço principal do cliente. Se for necessário, você pode clicar no campo de endereço e fazer alterações. Se for preciso adicionar um novo endereço, selecione o botão "Novo Endereço".
    • Tipo de Serviço - Escolha o tipo de serviço a ser realizado.
    • Protocolo - Insira o protocolo fornecido pela empresa.
    • N° OS Algar - Informe o número da ordem de serviço fornecido pela empresa.
    • Comodatar Modem - Marque essa opção se os seriais da OS devem gerar obrigatoriamente um comodato.
  • 3. Na tela de ordem de serviço, você encontrará as seguintes abas para gerenciar as informações:
    • Produtos - Utilize esta aba para listar os produtos utilizados ou consumidos durante a execução da OS.
    • Coleta - Na aba "Coleta", você pode registrar a coleta de modems comodatados e não comodatados do cliente. Se o modem estiver em comodato, ele aparecerá nesta seção. Caso seja necessário coletar um modem durante o serviço, basta clicar em "Coletar". Se precisar coletar modems com números de série que não estejam cadastrados no sistema, selecione o produto, insira o número de série e clique em "Adicionar".
    • Serviços - Nesta aba, é possível especificar os serviços realizados como parte da ordem de serviço.
    • Observações - Utilize esta seção para adicionar quaisquer observações relevantes ou notas adicionais relacionadas à ordem de serviço.
  • Essas abas auxiliam na organização e registro de informações detalhadas relacionadas à execução da ordem de serviço. Certifique-se de preencher as informações necessárias em cada aba, conforme a necessidade da OS em questão.
  • 4. Após preencher as informações haverão 3 tipos de possibilidade:
    • Salvar - Salva a OS, mas não fecha ela, ou seja, ainda será possível alterar/adicionar algumas informações, porém não será possível alterar o cliente, loja e técnico. A OS ficara com o status “Aberta”
    • Fechar - Não fechar uma OS, ela ficará com o status “Fechada”.
    • Cancelar / Cancelar Alterações - Se a Os ainda não foi gravada pela primeira vez, ainda é possível cancelá-la clicando no botão de cancelar. Agora se a OS já foi gravada, não é possível cancelar, quando estiver alterando uma OS já gravada que está com status “Aberta” o botão será “Cancelar alterações” nesse botão ele cancela todas as alterações feitas na OS aberta.

Transferência

Uma tela de transferência de mercadoria é uma interface de usuário projetada para facilitar o processo de transferir produtos ou mercadorias de um local para outro dentro de uma organização. Essa funcionalidade desempenha um papel essencial na gestão de estoque e na otimização da cadeia de suprimentos.

  • Funções da Tela de Transferência de Mercadorias
  • 1. Filtro - O filtro permite a busca de uma transferência específica pelo número. Ao clicar no ícone do filtro, é possível filtrar informações como a origem do estoque (a loja de onde a mercadoria saiu), o destino do estoque (a loja para a qual a mercadoria foi encaminhada) e o status da transferência.
  • 2. Existem quatro status diferentes:
    • Concluída - Indica que a transferência foi finalizada, o que significa que os produtos foram entregues à loja de destino e a transferência não pode mais ser alterada.
    • Pendente - Os itens foram enviados, mas ainda não chegaram à loja de destino.
    • Cancelada - Os produtos foram enviados, mas, por alguma interferência externa, a empresa decidiu cancelar o envio.
    • Extraviada - Esse status significa que algo aconteceu com a mercadoria no percurso, e não se sabe onde ela pode estar (pode ter sido roubada ou perdida no trajeto).
  • 3. Na parte de ações da tabela, você encontrará as seguintes funções:
    • Visualizar - Permite ver algumas informações detalhadas da transferência.
    • Alterar - Possibilita a alteração do CFOP de origem e destino da transferência.
    • Três Pontinhos - Ao clicar nessa opção, um submenu é aberto, oferecendo três escolhas: "Status", "Imprimir" e "Gerar NF-e". A função "Status" permite a alteração do status de uma transferência que não esteja marcada como "Concluída" ou "Cancelada". A opção "Imprimir" gera um comprovante da transferência, enquanto "Gerar NF-e" possibilita a geração de uma Nota Fiscal Eletrônica (NF-e).

Transferência - Novo

Para adicionar uma nova transferência, siga os passos abaixo:

  • 1. Clique no botão "Novo".
  • 2. Preencha os seguintes campos:
    • Origem - Listar - Define o que o sistema irá listar no campo "Loja do Estoque de Origem". Ele pode listar Lojas, Outros (Sub-Estoques/Técnico) e todos (Lojas + Outros).
    • Loja do Estoque de Origem - Escolha a loja de onde os produtos sairão.
    • CFOP de Origem - O CFOP (Código Fiscal de Operações e Prestações) determina o tipo da operação de transferência. Este é um campo fiscal, e a escolha do CFOP correto requer contato com o escritório contábil.
    • Destino - Listar - Define o que o sistema irá listar no campo "Loja do Estoque de Destino". Ele pode listar Lojas, Outros (Sub-Estoques) e todos (Lojas + Outros/Técnico).
    • Loja do Estoque de Destino - Escolha a loja para a qual os produtos serão encaminhados.
    • CFOP de Destino - O CFOP de destino é o código fiscal que determina o tipo da operação da transferência. Como no CFOP de origem, é um campo fiscal, e a escolha do CFOP correto requer contato com o escritório contábil.
  • 3. Na parte de produtos, selecione e informe os produtos que serão incluídos na transferência.
  • 4. Clique em "salvar" para gravar a transferência ou em "Cancelar" para cancelar a operação.Na parte de produtos, selecione e informe os produtos que serão incluídos na transferência.
  • 5. Após ter adicionado a transferência, ela ficará com o status "pendente" até que a loja de destino confirme que os produtos chegaram. Para alterar o status da transferência, clique nos três pontinhos na seção de ações e depois em "Status".
  • 6. Agora você pode confirmar, cancelar ou declarar a transferência como "extraviada".

Requisição - Novo

Para adicionar uma nova requisição, siga os passos abaixo:

  • 1. Clique no botão "Novo" na tela de requisição.
  • 2. Preencha os seguintes campos:
    • Urgência - Defina o nível de urgência da requisição como baixa, média ou alta.
    • Requisitado - Listar - Escolha o que o sistema irá listar no campo "Requisitado". As opções incluem Lojas, Outros (Sub-Estoques/Técnico) e todos (Lojas + Outros).
    • Requisitado - Escolha a empresa, sub-estoque ou técnico que enviará os produtos solicitados.
    • Requisitante - Listar - Escolha o que o sistema irá listar no campo "Requisitante". As opções incluem Lojas, Outros (Sub-Estoques/Técnico) e todos (Lojas + Outros).
    • Requisitante - Escolha quem está solicitando os produtos.
  • 3. Informe os produtos que serão requisitados.
  • 4. Após preencher todas as informações, clique em "Salvar" para salvar a requisição, ou em "Cancelar" para não salvá-la.
  • 5. A requisição será gerada e, se o requisitado enviar todos os itens solicitados, a requisição ficará com o status "Transferidos". Se apenas alguns itens forem enviados, o status será "Transf. Parcial", e se nenhum item for enviado, o status será "Negado".
  • 6. As requisições com status "Transferidos" ou "Transf. Parcial" gerarão uma transferência.

Comodato - Funções

A tela de comodato é uma interface de usuário que facilita o gerenciamento de bens ou ativos em regime de comodato, ou seja, itens emprestados temporariamente a terceiros, geralmente clientes. Aqui estão algumas das funções disponíveis:

  • 1. Filtro - Você pode usar o filtro para pesquisar comodatos com base em informações como loja, cliente, produto e vendedor. O sistema carrega automaticamente as informações a partir da data de "Início até hoje", mas você pode alterá-la clicando no botão "início até hoje". Após inserir as informações desejadas, clique em "Filtrar". Além disso, você pode realizar pesquisas rápidas usando o campo de pesquisa para informações como número de série, nome do cliente e nome do produto.
  • 2. Abas da tela de comodato - Esta tela possui três abas: "Todos", "Pendentes" e "Devolvidos".
    • Todos - Nesta aba, você encontrará todos os comodatos, incluindo os pendentes e os já devolvidos.
    • Pendentes - Aqui ficam apenas os comodatos que ainda não foram devolvidos.
    • Devolvidos - É onde ficam registrados os comodatos que já foram devolvidos.
  • 3. Três pontinhos - Dependendo do status do comodato, ao clicar nesta opção, diferentes ações estarão disponíveis:
    • Para comodatos devolvidos, a única opção será "Imprimir".
    • Para comodatos não devolvidos, você terá a opção de "Devolver" para registrar a devolução individualmente ou a opção "Imprimir" para imprimir o comodato.
  • 4. Devolver múltiplos comodatos - Selecione os comodatos não devolvidos e clique em "Devolver" para registrar a devolução de vários comodatos de uma só vez.

Comodato - Novo

Para adicionar um comodato, siga os passos abaixo:

  • 1. Clique em "Novo" na tela de comodato.
  • 2. Preencha as seguintes informações:
    • Loja - Informe a loja que é responsável pelo comodato.
    • Data - Insira a data do comodato.
    • Técnico - Indique o técnico responsável pelo comodato.
    • Cliente - Especifique o cliente que solicitou o comodato.
    • Endereço - O endereço será preenchido automaticamente com base nas informações do cadastro do cliente. Caso seja necessário adicionar endereços secundários, clique em "Novo Endereço" após selecionar o cliente e preencha as informações necessárias.
  • 3. A tela de comodato possui duas abas:
    • Aba de Produtos: Aqui, informe os produtos que serão incluídos no comodato.
    • Aba Adicional: Caso haja observações ou informações adicionais a serem registradas, este é o local apropriado.
  • 4. Após preencher todas as informações, clique em "Salvar" para gerar o comodato ou em "Cancelar" para abortar o processo.

Movimentação Avulsa - Funções

A movimentação de estoque desempenha um papel crucial no gerenciamento do fluxo de mercadorias e materiais dentro de uma organização. Essa funcionalidade lida com a entrada e saída de itens no estoque, bem como todas as transações relacionadas a essas operações.

  • 1. Filtro A- través do filtro na tela de movimentação de estoque, é possível filtrar as informações por ID, produto ou histórico, tornando mais fácil encontrar transações específicas.
  • 2. Ações - Na parte de ações, você tem acesso a algumas opções:
    • Visualizar - Ao clicar no ícone do olho, você pode verificar o conteúdo detalhado da movimentação de estoque, incluindo informações sobre os produtos envolvidos e as quantidades.
    • Editar - É importante observar que não é possível editar movimentações de estoque diretamente. Caso ocorra algum erro ou lançamento incorreto, é recomendável corrigir o problema com um lançamento reverso ou, se necessário, excluir a movimentação e registrá-la novamente corretamente.
    • Três pontinhos - Ao clicar nos três pontinhos, você pode escolher entre duas opções:
      • Excluir: Permite excluir uma movimentação de estoque, caso seja necessário.
      • Ver Log de Atividades: Essa opção exibe o histórico de atividades relacionadas à movimentação, mostrando todas as ações realizadas, como quem fez a movimentação, quando e quais alterações foram feitas.

Movimentação Avulsa - Novo

Para adicionar uma movimentação de estoque, siga estas etapas:

  • 1. Na tela de movimentação de estoque, clique no botão "Novo".
  • 2. Preencha as seguintes informações:
    • Produto: Selecione o produto que está envolvido na movimentação.
    • Loja: Especifique a loja relacionada a essa movimentação.
    • Histórico: Descreva o motivo da movimentação, como "Recebimento de fornecedor", "Transferência para outra filial" ou qualquer explicação apropriada.
    • Quantidade: Indique a quantidade de produtos envolvidos na movimentação. Este campo pode ser usado para especificar a quantidade que entra ou sai do estoque, dependendo da natureza da movimentação.
    • Valor: Registre o valor associado a essa movimentação, se necessário. Isso pode ser relevante para fins contábeis.
    • Movimentação: Escolha se esta é uma entrada ou saída de produtos do estoque.
  • 3. Após preencher todas as informações necessárias, clique em "Salvar" para registrar a movimentação no sistema ou clique em "Cancelar" para descartar as informações e não salvar a movimentação.

    • Três pontinhos: Ao clicar nos três pontinhos, é possível excluir o registro do serial ou visualizar o log de atividades relacionadas a ele.

Meu caixa - Funções

A tela de caixa desempenha um papel crucial na gestão financeira de uma empresa. Aqui estão as funções disponíveis:

  • 1. Novo - Permite lançar um novo movimento de caixa, sendo um atalho para adicionar um lançamento financeiro.
  • 2. Filtro - Ao clicar no botão "Filtro", você pode filtrar vendas e lançamentos por loja e data, facilitando a análise de transações financeiras.
  • 3. Acesso Rápido - Apresenta atalhos para telas relacionadas e uma tela de transferência de valores:
    • Transferências: Permite a transferência de valores entre caixas de lojas e contas correntes de lojas. Informe os detalhes e clique em "Salvar".
    • Vendas: Acesso rápido à tela de vendas.
    • Pré-Vendas: Atalho para a tela de pré-vendas.
    • Relatórios: Acesso rápido à tela de relatórios.
  • Detalhe: O acesso aos atalhos depende da permissão de acesso a telas específicas.
  • 4. Saldo Anterior - Mostra o saldo do caixa antes do período selecionado. O sistema, por padrão, exibe o mês inteiro, mas você pode ajustar o período clicando no botão "Filtro" e alterando as datas.
  • 5. Entradas - Exibe o total de receitas registradas no caixa durante o período escolhido.
  • 6. Saídas - Mostra o total de despesas registradas no caixa durante o período selecionado.
  • 7. Saldo Atual - Indica o saldo atual do caixa.
  • 8. Vendas Abertas - Apresenta as pré-vendas não finalizadas do período. Cada pré-venda oferece as opções:
    • Finalizar: Permite a conclusão da pré-venda, abrindo a tela de forma de pagamento para continuar a venda.
    • Três Pontinhos: Ao clicar nessa opção, você pode visualizar o conteúdo da venda e, se necessário, excluí-la.
  • 9. Últimos Lançamentos - Exibe um resumo dos movimentos de caixa e vendas lançados no período. Ao lado do título "Últimos Lançamentos", há um botão de três pontos com essas funções:
    • Exibir Resumo: Mostra um resumo de todos os lançamentos, incluindo o saldo anterior, entradas, saídas e o total de cada forma de pagamento.
    • Atualizar: Atualiza as informações, buscando por novos lançamentos.
  • Cada Lançamento possui suas próprias opções nos três pontinhos com o seguintes:
    • Editar: Permite a alteração de informações de um determinado movimento de caixa. Não é possível alterar vendas.
    • Visualizar: Exibe o conteúdo de um documento (venda ou movimentação de caixa) específico.
    • Excluir: Apaga o movimento ou venda selecionada.
    • Log de Atividades: Mostra as alterações de um movimento, caso existam registros de atividades.

  • 8. Últimos Lançamentos - Lista as movimentações bancárias mais recentes, permitindo visualizar as informações das transações. Você pode atualizar a lista de movimentações clicando no botão "Atualizar" localizado na parte superior.

Acessos

A tela de "Acessos" oferece aos administradores e gestores a capacidade de controlar permissões e configurações específicas para cada usuário. Ela é dividida em duas etapas: cadastro de níveis de acesso e cadastro de usuários, permitindo a personalização das permissões de acordo com as necessidades da empresa.

  • Cadastro de Níveis de Acesso: A primeira etapa envolve o cadastro de níveis de acesso. Um nível de acesso define as permissões que serão atribuídas aos usuários. Nova Venda

A tela "Nova Venda" permite que os funcionários da loja registrem transações de venda de maneira eficiente e simplificada. Siga os passos a seguir para realizar uma venda:

    • Na tela de venda, preencha as seguintes informações:

    • Vendedor: Escolha o vendedor que está realizando a venda (o sistema geralmente preenche o nome do vendedor com base no cadastro do funcionário logado).
    • Cliente: Digite o nome do cliente que está fazendo a compra.
    • Endereço: O sistema preenche automaticamente com o endereço principal do cliente selecionado. Se o cliente tiver outros endereços, é possível adicionar um novo endereço clicando em "Novo Endereço" após a seleção do cliente.

  • A parte inferior da tela possui as seguintes abas:

    • Entradas: Use esta aba para registrar devoluções ou vendas com trocas. Adicionar um produto aqui indica que o produto está retornando para o estoque e será descontado do valor da venda.
    • Serviços: Esta aba permite adicionar serviços realizados na venda.
    • Adicional: Insira observações relevantes sobre a venda, como o local onde o cliente retirará o produto.
  • 3. Descontos:
    • À direita da tela, você pode adicionar descontos ou acréscimos à venda. Selecione a opção desejada e especifique se o valor será aplicado como porcentagem ou valor fixo. Clique em "Aplicar" para incluir o desconto ou acréscimo na venda.
  • 4. Pagamentos:
    • Para finalizar a venda, clique em "Formas de Pagamento". Aqui, é possível adicionar uma ou várias formas de pagamento. Se o cliente utilizar várias formas de pagamento, insira o valor em cada forma até que o total seja atingido.

    • Após inserir as informações da forma de pagamento, clique em "Finalizar". O sistema gerará o comprovante de venda.

    • Alguns campos na tela de vendas possuem um ícone de lápis com uma seta para baixo. Ao clicar nesse ícone, um submenu é exibido com opções como "Novo", "Cadastro Completo" e "Limpar". Essas opções permitem a adição rápida de clientes ou acesso direto a informações completas do cadastro.

Pré-Venda

A tela de Pré-Venda é uma ferramenta essencial para os vendedores registrarem e processarem pedidos de clientes antes que a transação de venda seja finalizada. Embora siga o mesmo modelo da Venda Balcão, a principal diferença é que a Pré-Venda não é finalizada na própria tela. Em vez disso, a Pré-Venda é registrada e posteriormente finalizada na tela de Caixa.

  • 1. Na tela de Pré-Venda, preencha todas as informações relevantes da venda, incluindo detalhes do cliente, produtos, quantidade, preços e quaisquer descontos aplicáveis.
  • 2. Após preencher todos os dados da Pré-Venda, você não finaliza a venda nesta tela. Em vez disso, a venda é salva temporariamente e aguarda finalização na tela de Caixa.
  • 3. Vá para a tela de Caixa quando estiver pronto para concluir a transação. Na tela de Caixa, você seleciona a Pré-Venda correspondente e prossegue para finalizá-la.
  • 4. No processo de finalização, você escolhe a forma de pagamento, confirma os valores e, em seguida, clica em "Finalizar Venda" para concluir a transação e registrar o pagamento.

Transferência

Uma tela de transferência de mercadoria é uma interface de usuário projetada para facilitar o processo de transferir produtos ou mercadorias de um local para outro dentro de uma organização. Essa funcionalidade desempenha um papel essencial na gestão de estoque e na otimização da cadeia de suprimentos.

  • Funções da Tela de Transferência de Mercadorias
  • 1. Filtro - O filtro permite a busca de uma transferência específica pelo número. Ao clicar no ícone do filtro, é possível filtrar informações como a origem do estoque (a loja de onde a mercadoria saiu), o destino do estoque (a loja para a qual a mercadoria foi encaminhada) e o status da transferência.
  • 2. Existem quatro status diferentes:
    • Concluída - Indica que a transferência foi finalizada, o que significa que os produtos foram entregues à loja de destino e a transferência não pode mais ser alterada.
    • Pendente - Os itens foram enviados, mas ainda não chegaram à loja de destino.
    • Cancelada - Os produtos foram enviados, mas, por alguma interferência externa, a empresa decidiu cancelar o envio.
    • Extraviada - Esse status significa que algo aconteceu com a mercadoria no percurso, e não se sabe onde ela pode estar (pode ter sido roubada ou perdida no trajeto).
  • 3. Na parte de ações da tabela, você encontrará as seguintes funções:
    • Visualizar - Permite ver algumas informações detalhadas da transferência.
    • Alterar - Possibilita a alteração do CFOP de origem e destino da transferência.
    • Três Pontinhos - Ao clicar nessa opção, um submenu é aberto, oferecendo três escolhas: "Status", "Imprimir" e "Gerar NF-e". A função "Status" permite a alteração do status de uma transferência que não esteja marcada como "Concluída" ou "Cancelada". A opção "Imprimir" gera um comprovante da transferência, enquanto "Gerar NF-e" possibilita a geração de uma Nota Fiscal Eletrônica (NF-e).

    • Loja do Estoque de Destino - Escolha a loja para a qual os produtos serão encaminhados.
    • CFOP de Destino - O CFOP de destino é o código fiscal que determina o tipo da operação da transferência. Como no CFOP de origem, é um campo fiscal, e a escolha do CFOP correto requer contato com o escritório contábil.

Acessos

Acesso e Controle de Usuários

Acessos

A tela de "Acessos" oferece aos administradores e gestores a capacidade de controlar permissões e configurações específicas para cada usuário. Ela é dividida em duas etapas: cadastro de níveis de acesso e cadastro de usuários, permitindo a personalização das permissões de acordo com as necessidades da empresa.

  • Cadastro de Níveis de Acesso: A primeira etapa envolve o cadastro de níveis de acesso. Um nível de acesso define as permissões que serão atribuídas aos usuários. Isso permite agrupar usuários com necessidades semelhantes de acesso em um mesmo nível. Os administradores podem personalizar as permissões de acordo com os privilégios que desejam conceder a cada nível. Também é possível alterar um nível existente clicando no ícone de alterar dele.
  • Cadastro de Usuários: A segunda etapa envolve o cadastro de usuários. Durante o cadastro de um usuário, os administradores associam o usuário a um nível específico e, dessa forma, herdam as configurações de acesso definidas para aquele nível.

Níveis de Acessos

Para adicionar um novo nível de acesso, siga os seguintes passos:

  • Configuração de Permissões de Nível de Acesso: Ao criar um novo nível de acesso, você precisará definir o nome do nível e, o mais importante, as permissões para esse nível. As permissões podem ser configuradas por módulos, o que permite especificar quais áreas do sistema um usuário com esse nível de acesso pode ou não acessar. Os módulos comuns podem incluir:
    • Cadastro
    • Estoque
    • Financeiro
    • Vendas e Caixa
  • Dentro de cada módulo, é possível acessar as seguintes opções:
    • Recolher: Fecha todos os módulos, o que pode ser útil para simplificar a visualização.
    • Expandir: Expande todos os módulos para facilitar a configuração.
    • Marcar Todos: Marca todas as permissões disponíveis, liberando todas elas para o nível de acesso.
    • Desmarcar Todos: Remove todas as marcações, bloqueando o acesso a todas as permissões disponíveis.
  • Após liberar ou bloquear as permissões de sua escolha, clique em "Salvar" para confirmar as configurações. Isso criará o novo nível de acesso com as permissões personalizadas.

Usuário

Na seção de gerenciamento de usuários, logo abaixo dos níveis, você pode encontrar as seguintes funções de filtro:
